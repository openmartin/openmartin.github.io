workflow 推荐

Posted on 2020-10-27 11:00 in Java

workflow

这里的 workflow 工作流 仅仅指针对企业的业务流程的工作流，任务调度之类的不包含在内

从最初的 jBPM， 工作流在不停的分支，目前主流的是 Flowable 和 Camunda

jBPM4 => jBPM6
      => Activiti5 => Activiti6
                   => Flowable
                   => Camunda

jBPM4 => Activiti5 => Flowable 这一条线算是一脉相承，虽然名字变了，但是还是同一拨人

Camunda

Camnuda 也是一个很优秀的工作流，继承 Activiti5 开发，企业版有一些高级的功能，比如 BPMN Heatmaps 查看热点的区域。
