pandoc

pandoc 是一个haskell语言开发的文档格式转换工具，支持非常丰富的格式，覆盖了常见的所有格式。使用起来非常方便。

安装

# mac osx
brew install pandoc
# ubuntu
sudo apt-get install pandoc

格式转换

reStructuredText to markdown

pandoc -f rst --toc small_tip.rst -t markdown_github -o small_tip.md
